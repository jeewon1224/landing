## Feature
- 메인 배너 슬라이드 배치
- swiper의 slideChangeTransitionEnd 옵션을 활용하여 메인 배너 슬라이드의 백그라운드 변경 이벤트 생성
- 동적인 최신 앨범 슬라이드
- iframe 태그의 src를 활용한 비디오 슬라이드
